Trooper

Trooper is a simple helper for git uploads. You no longer go through the process of adding, creating commits, and pushing... Just run trooper!

Usage

The usage is simple just:

trooper [git_username] [repository_name]

Installation

There is a simple list of commands for you to run for installation...
1. Download Trooper source.
2. Unzip.
3. Run open /usr/local in the command line.
4. Drag Trooper folder into the newly opened finder window.
5. Run these commands in the command line:

cd ~
touch .bash_profile # If you do not have one.
echo "export PATH=$PATH:/usr/local/trooper/bin" >> .bash_profile

And you are done!
