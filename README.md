# Trooper
Trek back to your repo...

# Installation
1. `cd /usr/local`
2. `git clone https://github.com/blubrackets/trooper`
3. `cd ~`
4. `echo "export PATH=$PATH:/usr/local/trooper/bin" >> .bash_profile`
5. Restart command line...
6. `trooper --help`

# Usage
`trooper [git_username] [repository_name]`
